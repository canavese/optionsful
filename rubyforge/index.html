About

Provide dynamic information about resources via the OPTIONS' HTTP method on a RESTful design over the Ruby on Rails framework.

Installation

The optionsful gem is currently hosted at RubyGems repository. To install it, run: 

          $ gem install optionsful
  1. Change directory to your Ruby on Rails web application
  2. Add gem dependency to config/environment.rb:
    3.             config.gem 'optionsful'
  3. Have fun!

Usage examples:

The optionsful gem is currently hosted at RubyGems repository. To install it, run:

Note: notice the empty line, which is part of the HTTP protocol
